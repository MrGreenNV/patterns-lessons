Шаблон: Абстракная фабрика (Abstract Factory)
Цель: Создание интерфейса, для создания множества взаимосвязанных или взаимозависимых объектов, без жесткой привязки
к конкретным классам.
Применение: Для создания множеств взаимосвязанных объектов.
Пример использования:
- система не должна зависить от метода создания, компоновки и предоставления входящих в неё объектов;
- входящие взаимосвязанные объекты должны использоваться вместе;
- система должна конфигурироваться одним из множеств объектов, из которых она состоит;
- нам необходимо предоставить множество объектов, раскрывая только их интерфейсы, но не реализацию.
